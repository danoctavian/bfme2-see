Crash Troubleshooting

Crashing is the process of a program shutting down unexpectedly, usually leaving you back at your Windows Desktop.  A game will typically crash as such when there are conflicts between its files or requirements and the hardware in the computer it is trying to run on. The following section of the help troubleshooting manual will guide you through common issues that can cause Crashes while trying to play. There are a few common crash types which point to different causes.
