Transfer or File Error During Installation

Transfer errors during installation can be caused by a number of conflicts. If you are experiencing a Transfer or File error during installation:

  • End all background tasks to prevent conflicts with other games

Click here for steps on ending your background tasks

  • Empty your Temp folder to prevent file conflicts

Click here for instructions on emptying your Temp files

  • If you have a secondary CD/DVD-ROM drive, try installing the game from that drive

  • The issue could be caused by an issue with your CD/DVD-ROM drive

Click here for CD/DVD-ROM troubleshooting steps
