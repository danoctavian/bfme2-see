Configuring Routers and Firewalls

Each game which has an online component requires certain 'Ports' to be available to connect to that feature. [Ports are numbered connections that a computer uses to sort types of network traffic.] By default, all ports to the Internet, and most LAN ports are closed by your router so that traffic cannot flow through them. By opening just the ports required by a game, you will be able to connect to the online service while still keeping unwanted traffic closed. Please select your specific router type for basic instructions on opening ports:

   

To configure a SMC Model# SMC7004VWBR Wireless Barricade Router:

  1. Before you attempt to configure your router please visit www.smc.com and download the latest firmware update.
  2. To access the SMC Wireless Barricade's management interface, open your web browser (Microsoft Internet Explorer, Netscape Navigator, etc.). In the address box enter http://192.168.2.1 which is the default address for the SMC7004VWBR Wireless Router. Then click [Login]. (By default there is no password)
    1. Note : If you changed the router's IP address or specified a password, then enter the new address in your web browser and specified password at the login screen. For more information, please consult your SMC User's Guide.
  3. At the home page of the SMC management interface, click Advanced Setup.
  4. On the left hand side of the Advanced Setup screen click on NAT and then select Special Application from the sub menu.
  5. On the Special Application screen you will need to set up an individual rule for each computer. For each computer define the following settings.
    1. TRIGGER PORT : Enter a port number that is used to initiate the connection. Do not use a standard port.
      • Example : You can use 27901 for the first computer and 27902 for the second and so on. The trigger port must be unique for each computer.
    1. TRIGGER TYPE : Select UDP, the protocol used by most games.
    2. PUBLIC PORT : Enter the same port number here that you entered in the Trigger Port field for each computer.
    3. PUBLIC TYPE : Select UDP, the protocol used by most games.
    4. ENABLED : Check this box to enable the settings you entered for each computer.
      • Note : Write down each port number used in step 5, as you may need this to configure the in-game options of the game you are trying to play online.

 

To configure a Linksys Model# BEFSR11 or BEFSR41 Router:

  1. Before you attempt to configure your router please visit www.linksys.com and download the latest firmware update.
  2. To access the web based utility of your Linksys Router, open your web browser (Microsoft Internet Explorer, Netscape Navigator, etc.). In the address box enter http://192.168.1.1 which is the default address for the Linksys BEFSR11/BEFSR41 Router. A password request page window will pop up. Leave the user name blank and enter admin (all lower case, no capitol letters) in the password field. Then, click OK.
    1. Note : If you changed the routers IP address, or specified a user name or new password, then enter the new IP address in your web browser and/or enter the specified user name and new password at the password request window. For more information, consult your Linksys User's Guide.
  3. From the main setup page click on the Advanced tab, then the Forwarding tab and finally at the bottom of the screen click on Port Triggering.
  4. On the Port Triggering screen you will need to set up an individual rule for each computer connected to your Router/Firewall. For each computer define the following settings.
    1. APPLICATION NAME : Enter the name of the game you are trying to configure the router for.
    2. TRIGGER PORT RANGE : Enter a port number that is used to initiate the connection. Do not use a standard port.
      • Example : You can use 27901 for the first computer and 27902 for the second and so on. Be sure to enter the same port number in both fields under this heading.
    1. INCOMING PORT RANGE : Enter the same port number here that you entered in the Trigger Port Range field for each computer. Be sure to enter the same port number in both fields under this heading.
      • Note : Write down each port number used in step 4, as you may need this to configure the in-game options of the game you are trying to play online.
