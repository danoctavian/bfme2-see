Finding the Minimum System Requirements

The minimum requirements can be found in any of five different places for any EA game.

 

The first, and easiest to find is on the game box, itself.

 

    1. Locate the System Requirements sticker.
      • The left side of the sticker lists all the hardware requirements.
      • The right side of the sticker lists the video requirements and the supported video chipsets.

 

The second way to find the requirements is to check the Install guide.

 

    1. Included in your game box are a Manual and a white Install guide. Locate the Install Guide.
    2. Locate the System Requirements page. This is usually the first page.
      • The Minimum Configuration that can play the game is listed here.

 

The third way to get the requirements, if the box and manual are unavailable, is to open the Readme file for the game.

 

    • The Readme is most easily found through the Start Menu location for the game. The location for these is different between EA Games and EA Sports titles, and if you chose a different location, you will need to go to that custom location instead. The System Requirements are typically the first section after the Table of Contents, but check the Table of Contents to verify its location.
      • EA Games default Readme location:
        1. Click on the Start button.
        2. Go to All Programs.
        3. Go to EA GAMES.
        4. Go to the game title.
        5. Click on the Readme.
      • EA Sports default Readme location:
        1. Click on the Start button.
        2. Go to All Programs.
        3. Go to EA SPORTS.
        4. Go to the game title.
        5. Click on the Readme

 

The fourth place to check is the EAsy Info tool. The requirements listed here are the least complete, so try to use one of the other methods if possible. The only benefit of using the EAsy Info tool is that you get an instant comparison of your system and the requirements. However, it cannot usually give you full video card information and sometimes cannot fully detect all hardware configurations.

    The EAsy Info tool has a similar location to the Readme file; it's location is different between EA Games and EA Sports. If you chose a custom install location, please refer to that location instead.

      • EA Games default EAsy Info location:
        1. Click on the Start button.
        2. Go to All Programs.
        3. Go to EA GAMES.
        4. Go to the game title.
        5. Click on EAsy Info.
      • EA Sports default EAsy Info location:
        1. Click on the Start button.
        2. Go to All Programs.
        3. Go to EA SPORTS.
        4. Go to the game title.
        5. Click on EAsy Info
         

The Game Requirements are listed on the right side of the window. The only System Requirements information listed here is the Operating System, necessary system memory (RAM), CPU speed, DirectX version, and display memory. For further information, you must use one of the other methods for checking the System Requirements.

 

If none of the above is available (if this is a pre-purchase inquiry, etc.) you may also search for the system requirements on our Support site.

 

    1. Go to our support site at http://support.ea.com.
    2. Click on Technical Support.
    3. In the Platform drop down menu, choose PC.
    4. Choose your game name from the Product menu.
    5. In the search text box type a keyword, in this case it would be requirements.
    6. Click the Search button.
    7. A list of related answers will appear. Click on the question in the Summary field to view the answer. The question in this case would be 'What are the system requirements?'
    8. This answer is most useful as a reference to check your system before purchasing the game.
