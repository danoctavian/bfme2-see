'Cannot locate CD/DVD-ROM' or 'Please insert correct CD/DVD'

These errors are typically caused by conflicts reading the disk, which can be a conflict with the disk or the drive itself. It can also be caused by conflicts with other files and programs that interrupt the proper reading of the disk.

 

  • End all unnecessary background tasks to prevent programs from conflicting with the game and the reading of the disk

Click here for steps on ending background tasks

  • Empty your Temp folder to prevent file conflicts when starting up the game

Click here for steps on emptying your Temp folder

  • If you have a secondary disk drive, try using this drive instead

  • Try CD/DVD-ROM troubleshooting steps to prevent read-related errors

Click here for CD/DVD-ROM troubleshooting steps

  • If the issue occurs during installation, be sure to complete the basic installation troubleshooting steps

Click here for help with installation issues
